## 객체 
    로또기계, 로또, 사용자

-------------------

## main함수 동작
>1. 구입금액 입력(사용자)
>2. 로또 n개 생성
>3. LottoMachine에 당첨번호 입력
>4. 보너스 번호 입력
>5. 로또 n개(사용자)랑 당첨번호(로또기계) 비교
>6. 당첨 결과 출력
-------------------
## [로또기계]
### 클래스 변수
* 당첨 로또번호
* 당첨 통계

### 매서드
* 사용자 로또 list랑 당첨 로또랑 비교 = 당첨 통계 생성
* 당첨 통계 출력


-------------------
## [로또]
### 클래스 변수
* 숫자들 list
### 매서드
* 입력받은 숫자가 올바른지 확인하는 매서드들..
-------------------
## [사용자]
### 클래스 변수
* 구매한 로또 List
* 돈
### 매서드
* 로또를 사는 행위 (랜덤번호 생성)
